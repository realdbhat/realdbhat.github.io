只要我拷贝的足够深，就能得到不同的你 ——原型模式

1.浅拷贝

实现基本类型和String的拷贝，引用类型只拷贝地址。

2.深拷贝

I.重写clone方法

这个方法不太好用，因为只要一个类需要深拷贝，那么所有和这个类组合聚合的类都必须实现深拷贝。

#####II.通过序列化

只要在需要深拷贝的对象类中实现深拷贝方法，并给这个类和所有和这个类进行组合聚合的类打上Serializable接口标记就能实现对象的深拷贝了。
